# Pain Point Studio

A structured research platform for developers to validate product ideas before writing code. From hypothesis to first feedback in hours, not weeks.

## Quick Start

**Prerequisites:** Node.js 22+, pnpm 10+, Docker Desktop

```bash
pnpm install
cp .env.local.example .env.local
pnpm run:dev
```

- **App**: http://localhost:3000
- **Supabase Studio**: http://localhost:54323

> Docker must be running before `pnpm run:dev`.
